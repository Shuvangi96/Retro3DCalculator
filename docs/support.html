Welcome to the support page for RetroCalculator, a realistic 3D-skinned calculator with advanced memory, percentage, and replay capabilities.

Core Features

  • Memory operations (M+, M-, MRC)
  • Constant operation mode (e.g., 10 + 2 = = =)
  • Grand Total and Markup (MU)
  • Context-aware percentage logic
  • Replay and check mode for history review
  • Tactile visual + audio button feedback

Common Button Functions

  • M+: Adds display or result to memory
  • M-: Subtracts from memory
  • MRC: Recalls memory; double press clears it
  • MU: Markup pricing via percent or flat add
  • Check: Step through calculation history
  • Replay: Auto-play full calculation session
  • AC / ON: Full calculator + memory reset
  • CE: Clear current entry only
